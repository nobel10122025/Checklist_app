# Checklist_app

## Technologies used :
- HTML5
- CSS3
- Vanilla Javascript

## Exciting Features :
 - It has two themes (light and dark theme).
 - Wishes can be easily created, edited, and deleted.
 - Refreshing the page doesn't delete the wishes.
